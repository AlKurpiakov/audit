# Audit

Lightweight Linux server monitoring tool written in Bash.

## Features

- Real-time dashboard (TUI)
- CPU / Memory / Disk monitoring
- Port → process inspector
- Alert system with email notifications
- Background agent (systemd)
- ASCII graphs

## Usage
```bash
audit dashboard
audit monitor
audit ports
audit agent
```

## Installation

```bash
curl -sSL https://raw.githubusercontent.com/USER/audit/main/install.sh | bash
```
